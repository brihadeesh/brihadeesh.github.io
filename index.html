This is a static website and is intended to (eventually) be the home for my blog, Emacs configuration and work updates. I’ve called it my sourcehut site because I hope to eventually move to or host it there. This website uses etch with Hugo for the styling, and everything you see here was written in org-mode, exported to markdown using ox-hugo for better compatibility with Hugo - the default go-org parser doesn’t appear to support org-mode syntax in its entirety.
